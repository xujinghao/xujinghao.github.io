前言

面试碰到了CSS布局相关的,有相关Flex的部分,但是发挥不是太好。所以自己研究了一下具体的相关属性。具体的定义和发展没必要写直接进行实战。

Flex容器属性

flex的容器

基本css语法：

定义了一个flex容器，根据设值的不同可以是块状容器或内联容器。这使得直接子结点拥有了一个flex上下文。

flex-direction

flex-direction决定了flex项目的主方向。

基本css语法：

  • row 表示从左向右排列
  • row-reverse 表示从右向左排列
  • column 表示从上向下排列
  • column-reverse 表示从下向上排列

属性实战：

flex-wrap

默认的情况下，Flex项目都排在一条线（又称”轴线”）上。我们可以通过flex-wrap属性的设置，让Flex项目换行排列。

基本css语法：

  • nowrap(默认) 所有Flex项目单行排列
  • wrap Flex项目多行排列，按从上到下的顺序
  • wrap-reverse Flex项目多行排列，按从下到上的顺序

属性实战：

justify-content

基本css语法：

  • flex-start(默认)：从启点线开始顺序排列
  • flex-end：相对终点线顺序排列
  • center：居中排列
  • space-between：项目均匀分布，第一项在启点线，最后一项在终点线
  • space-around：项目均匀分布，每一个项目两侧有相同的留白空间，相邻项目之间的距离是两个项目之间留白的和
  • space-evenly：项目均匀分布，所有项目之间及项目与边框之间距离相等

属性实战：

align-items

基本css语法：

  • stretch(默认)：交叉轴方向拉伸显示
  • flex-start：项目按交叉轴起点线对齐
  • flex-end：项目按交叉轴终点线对齐
  • center：交叉轴方向项目中间对齐
  • baseline：交叉轴方向按第一行文字基线对齐

属性实战：


align-content
