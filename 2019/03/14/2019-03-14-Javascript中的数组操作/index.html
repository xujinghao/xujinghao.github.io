Javascript常见的数组操作

发表于 2019-03-14 | 分类于 Javascript ， js数组 | 阅读数：
字数统计： 5.8k | 阅读时长 ≈ 11 分钟.

前言

数组在Js中的地位至关重要，作为常见的数据类型，数组有很多方法，为了方便对数组进行操作。总结一下经常使用的几种数组方法

push

定义是：push()方法可向数组的末尾添加一个或多个元素，并返回新的长度。

  1. push()方法会向原来的数组中的末尾添加元素，一个或者多个元素；
  2. push()方法返回的值是添加元素之后的数组长度；
  3. push()方法会改变原来数组；

pop

定义是：pop()方法用于删除并返回数组的最后一个元素。

  1. pop()方法会删除原来的数组中的末尾的一个元素；
  2. pop()方法返回的值是原本数组中的最后一个元素，但是原数组如果是空数组的话返回值是undefined；
  3. pop()方法会改变原来数组, 但是数组如果是空数组不会改变；

unshift

定义是：unshift()方法可向数组的开头添加一个或更多元素，并返回新的长度。

  1. unshift()方法会向原来的数组中的开头添加一个元素；
  2. unshift()方法将把它的参数插入原数组的头部，并将已经存在的元素顺次地移到较高的下标处，以便留出空间；
  3. unshift()方法会改变原来数组；

shift

定义是：shift()方法用于把数组的第一个元素从其中删除，并返回第一个元素的值。

  1. shift()方法会删除原来的数组中的开头第一个元素,并且；
  2. shift()返回第一个元素的值，但是原数组如果是空数组的话返回值是undefined；
  3. shift()方法会改变原来数组,但是原数组如果是空数组不会改变；

splice

定义是：splice() 1.向数组中添加项目；2.从数组中删除项目然后返回被删除的项目。3.可以替换数组中的元素。

添加元素

  1. splice()方法会添加元素下标为2，删除0个元素，添加 "test";
  2. 由于没有删除元素，所以返回的值是一个空数组;
  3. splice()方法会改变原数组的下标

删除元素

  1. splice()方法会删除元素下标为2的元素，删除1个元素;
  2. 返回的值是原数组下标为2的元素;
  3. splice()方法会改变原数组的下标,删除之后数组的下标会少去一个

替换元素

  1. splice()方法会删除元素下标为2的元素，删除1个元素，添加 test;
  2. 所以返回的值是原数组下标为2的元素;
  3. splice()方法会改变原数组。

删除多个元素，并替换元素

  1. splice()方法会删除元素下标为2开始的元素，删除2个元素;
  2. 在数组的下标为2添加 "test"
  3. splice()方法会改变原数组。

concat

concat() 方法用于连接两个或多个数组。

连接参数到数组中

  1. concat()方法会将方法中的参数和数组进行联合
  2. concat()方法返回的值为连接之后的数组
  3. concat()方法不会改变原数组。

join

join() 方法用于把数组中的所有元素通过制定的分隔符放入一个字符串.

  1. 参数为空的话默认为’,’;
  2. join()方法不会改变原数组。

reverse

reverse() 方法用于颠倒数组中元素的顺序。

  1. join()方法改变原数组。

Map

map():对数组中每一项运行给定函数。返回每次函数调用的结果组成的数组。

  1. item是相当于数组中的元素遍历元素
  2. Map()方法不会改变原数组。

filter

filter():对数组中的每一项运行给定函数。返回该函数会返回true的项组成的数组。

数组去重

every

every():对数组中的每一项运行给定函数，如果该函数对每一项都返回true，则返回true 。

  1. 因为数组中的有元素没有大于1,所以返回false

some

some():对数组中的每一项运行给定函数，如果该函数对任一项返回true，则返回true 。

  1. 因为数组中的有一个元素大于3,所以返回true

find

find():方法返回传入一个测试条件（函数）符合条件的数组第一个元素 。

  1. 因为数组中的第一个大于1的元素是2,所以返回2

findIndex

findIndex():方法返回传入一个测试条件（函数）符合条件的数组第一个元素的下标 。

  1. 因为数组中的第一个大于1的元素下标是1,所以返回1

总结

数组中的操作还有很多，这里只是列出来常见的，常用的方法，通过基本的数组方法可以操作一些复杂的问题，从而减少代码量。
